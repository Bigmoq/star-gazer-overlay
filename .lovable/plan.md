

## خطة: داشبورد إدارة متكامل بالمحرك الأصلي — انقر على النجم وسجّله

### المشكلة الحالية
صفحة الأدمن تستخدم **iframe** من stellarium-web.org، مما يمنع التفاعل المباشر مع النجوم. التسجيل يتطلب نسخ/لصق روابط يدوياً.

### الحل
استبدال الـ iframe بالمحرك الأصلي (canvas + `useStellariumEngine`) في صفحة الأدمن، مع ربط النقر على أي نجم بنموذج تسجيل فوري.

### التدفق الجديد

```text
┌─────────────────────────────────────────────┐
│  صفحة الأدمن (محرك أصلي fullscreen)        │
│                                              │
│  [شريط علوي: لوحة الإدارة | النجوم | خروج]  │
│                                              │
│  ← المستخدم ينقر على نجم في السماء           │
│     ↓                                        │
│  تظهر لوحة تسجيل جانبية تحتوي:              │
│  • اسم النجم + بياناته (تلقائي من المحرك)    │
│  • حقل: اسم الشخص (مطلوب)                  │
│  • حقل: رسالة الإهداء                       │
│  • حقل: تاريخ الإهداء                       │
│  • زر "سجّل النجم"                          │
│     ↓                                        │
│  يُحفظ في Supabase → يظهر في قائمة النجوم   │
└─────────────────────────────────────────────┘
```

### الملفات والتغييرات

#### 1. `src/pages/Admin.tsx` — إعادة بناء كاملة
- إزالة الـ iframe و `IframeMask` بالكامل
- إضافة `<canvas>` + استخدام `useStellariumEngine` مع خيار `clickedStar`
- عند النقر على نجم (`clickedStar` يتغير) → فتح لوحة التسجيل تلقائياً مع البيانات المعبأة
- الإبقاء على: شريط علوي، قائمة النجوم، تعديل النجوم، تسجيل الخروج
- إضافة شريط أدوات سفلي بسيط (أبراج، غلاف جوي، تضاريس) مثل صفحة `/engine`

#### 2. `src/components/AdminRegisterPanel.tsx` — مكون جديد
- لوحة جانبية تظهر عند النقر على نجم
- تستقبل بيانات النجم من المحرك (`ClickedStarInfo`: الاسم، القدر، الإحداثيات)
- تولّد `code` تلقائياً من اسم النجم (مثل `HIP32349`)
- تبني `stellariumUrl` تلقائياً
- حقول: اسم الشخص، رسالة الإهداء، التاريخ
- عند الحفظ: `addStar()` → تحديث القائمة → toast نجاح مع الرمز
- التحقق من أن النجم غير مسجل مسبقاً (`findByCode`)
- تصميم glass-morphism متوافق مع باقي التطبيق

#### 3. `src/hooks/useStellariumEngine.ts` — تعديل بسيط
- إضافة `starIdentifier` (مثل `HIP 32349` أو `SAO 1818`) إلى `ClickedStarInfo` من designations
- إضافة خاصية `constellation` من المحرك إن توفرت

#### 4. إزالة/تبسيط `RegisterDrawer.tsx`
- لم يعد مطلوباً في صفحة الأدمن الجديدة (يُستبدل بـ `AdminRegisterPanel`)
- يبقى كملف لكن لا يُستخدم في Admin

### تفاصيل تقنية

- المحرك يوفر عبر `stel.change` + `core.selection` كل بيانات النجم المطلوبة
- `ClickedStarInfo` سيحتوي: `name`, `magnitude`, `ra`, `dec`, `identifier` (الكود مثل HIP/SAO/HD)
- `stellariumUrl` يُبنى: `https://stellarium-web.org/skysource/${identifier}?fov=0.5`
- التسجيل يمر عبر `addStar()` الموجود → Supabase `star_registry`
